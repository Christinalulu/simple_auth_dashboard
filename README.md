# Auth Dashboard

![Auth Dashboard Preview](/auth-dashboard/image/dashboard.PNG)

Et enkelt og brukervennlig autentiseringsdashboard bygget med React. Prosjektet demonstrerer grunnleggende autentiseringsfunksjonalitet med localStorage som database, og inkluderer en enkel oppgavebehandling.

## 📋 Funksjoner

- 🔐 **Brukerautentisering**
  - Registrering av nye brukere
  - Innlogging med eksisterende bruker
  - Data lagres i nettleserens localStorage

- ✅ **Oppgavebehandling (Dashboard)**
  - Legg til nye oppgaver
  - Marker oppgaver som fullført
  - Slett oppgaver
  - Enkelt og intuitivt grensesnitt

## 🛠️ Teknologier brukt

- **React** - JavaScript bibliotek for brukergrensesnitt
- **Tailwindcss** - Styling og layout
- **localStorage** - Klient-side datalagring
- **JavaScript (ES6+)** - Funksjonalitet og logikk
